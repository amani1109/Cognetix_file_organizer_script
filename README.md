# File Organizer Script

## Objective
Automatically organize files in a folder based on their file extensions.

## Features
- Organizes files by extension (.pdf, .jpg, .txt, etc.)
- Creates folders automatically
- Prevents file overwriting
- Handles unlimited files
- Simple console-based interface

## Requirements
- Python 3.x

## How to Run
1. Open terminal or command prompt
2. Navigate to the script location
3. Run: python file_organizer.py
4. Enter the full folder path when prompted

## Example
Before:
- notes.txt
- image.jpg
- report.pdf

After:
- txt_files/notes.txt
- jpg_files/image.jpg
- pdf_files/report.pdf
